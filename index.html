Introduction

IJIT (Interface for Just-In-Time translation) is an API which helps users to transform their [transition-system] based algorithms to a version that can operate directly on [Boolean Programs]. Said an API avoids converting a Boolean program to a transition system up front, as up-front conversion usually incurs state space blowup.

Get Started

}
